# rtg-tools

[English](README-zh.md) | 中文

`rtg` 当然**不是** `放射性同位素温差发电机`的缩写啦🤣。

它**不是**某个产品的`最佳实践`。我更愿意称之为`基准实践`—— 一方面它会比官方文档的 "入门章节" 这类`简单实践` 更深入一点、贴近实际需求；另一方面，如果你想要自己实现的话，起码不能（不该）比它实现的差。

它**是**一些开箱即用的中间件/数据库小工具的集合。

它**是**一组模板代码，注释完整，使用方式人性化。它能让你跳过大部分概念和语法细节，直接使用对应中间件/数据库。

## 使用

1. 找到你需要的文件夹。
2. 读Readme、注释和例子。
3. 复制代码到你本地，做你想做的任何事。

## 关于

关于为什么要写这个项目。

3年以前，我在工作中使用到 FastDFS。那时候没有官方 Python 客户端或者文档（现在也没有），开源社区只有一个 Python2 的客户端实现，但我们的业务正在往 Python3 上迁移。

所以我们重写了一个 Python3 版本的客户端。为了方便部署，我的同事把它上传到 pypi 上，之后我们就没有管过它了。

直到最近，我在互联网上发现了一些关于使用这个客户端的文章。老实说，当时写这个并没有花太多心思，现在看代码简直不忍直视。希望用它的小伙伴没有挨老板骂。

我能理解那种感觉，需要使用一个新玩意时，网上搜了一圈好像懂了又不知道到底做的对不对。所以我决定总结一下个人经验，告诉新人朋友在看完官方文档之后到底怎么样“真正的”去使用，或者为那些只想实现业务功能的人一个“不太坏”的实现。
